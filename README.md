# Steppe Coffee

Landing page for a specialty coffee shop chain. Warm earthy tones, elegant typography with Inter + Playfair Display fonts, scroll-driven animations.

## Tech Stack

- **Next.js 16** — App Router, Turbopack
- **Tailwind CSS v4** — `@theme` directive for custom design tokens
- **Framer Motion** — Scroll-triggered animations
- **Lucide React** — Icon system

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Project Structure

```
components/
  Header.tsx      — Fixed navbar with mobile hamburger menu
  Hero.tsx        — Full-screen hero with brand tagline
  About.tsx       — Brand story and values
  Menu.tsx        — Coffee menu showcase
  Locations.tsx   — Shop locations
  Footer.tsx      — Site footer with navigation
```
